# 瑞元工程塑料中文独立站

这是东莞市瑞元工程塑料有限公司中文独立站的静态版本，可直接上传到 GitHub 仓库根目录，并通过 Vercel 部署。

## 部署方式

1. 将本目录内所有文件上传到 GitHub 仓库：`luxylxjx-alt/123456`
2. 在 Vercel 新建项目，选择该 GitHub 仓库
3. Framework Preset 选择 `Other`
4. Build Command 留空
5. Output Directory 留空或使用根目录
6. 部署完成后绑定正式域名

## 当前包含页面

- 首页：`index.html`
- 阻燃 ABS 产品页：`products/flame-retardant-abs/index.html`
- 静态资源：`assets/`
- SEO 文件：`robots.txt`、`sitemap.xml`
- Vercel 配置：`vercel.json`
